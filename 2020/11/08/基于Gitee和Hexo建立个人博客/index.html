基于 Gitee 和 Hexo 建立个人博客

Posted on 2020-11-08 Edited on 2022-04-07 In Website

鉴于 GitHub Page 的不稳定和龟速，开始使用国内的 Gitee Page 来搭建个人博客。Gitee 是国内的一家开源代码托管平台，前不久 Gitee 也获得了工信部的支持，个人觉得 Gitee 还是不错的。

本文是基于 Gitee 和 Hexo 来搭建一个个人博客，网上有很多详细教程，这里主要记录一下搭建的流程以及后续问题，希望对你有帮助。
